Disneyland

I was part of the team that built the first version of the Mobile Ordering feature that enables guests of Disney World and Disneyland Resort to order food directly from their phones.

In this project, I've been able to play with various exciting integrations such as beacons, geofencing, and notifications.

I lead the team in charge of adding new features.

Disney World

I've worked on the dining reservations feature since 2016 when it was refactored and extracted into a library to be able to reuse it across different Disney apps.

I lead the refactoring from the entire feature from Java to Kotlin using the MVVM architectural pattern. Refactoring the code has helped us to increase the crash-free rate to 99% and made adding features faster.

Personal Projects

During my spare time I enjoy building apps that I use as a playground for the latest Android updates.

Next Idea

Next Idea is an Android application using Kotlin that allows users to save notes as ideas with different status and imagery.

I integrated Koin library as dependency injector framework, applied the MVVM pattern using Android Jetpack components and used the new app bundle feature to release and publish to the Play Store.

See it on Google Play

Bully Vets

The app was written using Dart, Swift, Kotlin. For the back-end I'm susing Cloud Storage for firebase.
