Hi, My name is Luis

I'm a Software Engineer at Disney with 5 years of experience building high scalable Android applications.

Mobile Ordering

I was part of the team that built the first version of the mobile ordering feature that enables guests of Disney World and Disneyland Resort to order food directly from their phones quickly.

In this project, I've been able to play with various exciting integrations such as beacons, geofencing, and notifications. I currently lead the team in charge of adding new features.

See in the Google Play Store

Dining Reservations

I've worked on this project since it was extracted into a library to be able to reuse it across different apps.

I lead the refactoring from the entire feature from Java to Kotlin using the MVVM architectural pattern. It has helped us to increase the crash-free rate to 99% and made adding features faster.

See in the Google Play Store

Personal Projects

During my spare time I enjoy building apps that I use as a playground for the latest Android updates.

Next Idea

Next Idea is an Android application using Kotlin that allows users to save notes as ideas with different status and imagery.

I integrated Koin library as dependency injector framework, applied the MVVM pattern using Android Jetpack components and used the new app bundle feature to release and publish to the Play Store.

See in the Google Play Store

Bully Vets

I developed an app using Flutter that allows bulldog owners to find veterinarians.

The app was written using Dart, Swift, Kotlin. For the back-end I'm susing Cloud Storage for firebase.

The app can be used on any Android or iOS devices but is currently only available in the Google Play Store

See in the Google Play Store

ABC Recycler View

Developed an Android library that provides the base classes to build recycler view adapter that makes easier to support multiple view types.

Released the library using jitpack.

See it in jitpack
